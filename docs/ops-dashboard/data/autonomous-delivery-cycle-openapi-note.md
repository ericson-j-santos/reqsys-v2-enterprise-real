# Nota API — Autonomous Delivery Cycle

Não há API externa nesta versão.

O contrato é baseado em arquivos estáticos JSON publicados pelo workflow:

- `autonomous-delivery-cycle-latest.json`
- `autonomous-delivery-cycle-next-increments.json`

Consumidores devem tratar ausência de arquivo, campos ausentes e fila vazia como fallback seguro.
