# Critérios de aceite — Autonomous Delivery Cycle

- [ ] Workflow `Autonomous Delivery Cycle` criado.
- [ ] Workflow aceita `dry_run` manual.
- [ ] Workflow possui schedule conservador.
- [ ] Merge exige `cycle:auto-merge-approved`.
- [ ] Merge exige `merge-queue:eligible`.
- [ ] Merge exige workflows obrigatórios verdes.
- [ ] Merge usa squash com SHA esperado.
- [ ] Pós-merge observa runs de `push`.
- [ ] Artifact `autonomous-delivery-cycle-report` é publicado.
- [ ] Próximos incrementos são capturados em fila report-only.
- [ ] Runbook, ADR, changelog e testes de contrato existem.
