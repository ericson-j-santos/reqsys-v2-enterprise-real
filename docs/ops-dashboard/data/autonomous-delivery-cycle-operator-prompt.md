# Prompt operador — Autonomous Delivery Cycle

```text
@GitHub

Valide o PR candidato ao Autonomous Delivery Cycle.

Critérios:
- aberto;
- não draft;
- mergeável;
- label merge-queue:eligible;
- label cycle:auto-merge-approved;
- todos os workflows obrigatórios verdes;
- baixo risco;
- sem falha pós-merge pendente.

Se tudo estiver verde, execute o fluxo governado apropriado.
Se não estiver, traga apenas blockers e links acionáveis.
```
