# Prioritization — Operational Health Dashboard

## Ordem de prioridade

1. PRs production-ready com CI verde.
2. PRs governance-only pequenos.
3. Correções de CI.
4. Fechamento de PRs superseded.
5. Recriação de branches antigas.
6. Incrementos experimentais.

## Regra prática

Em ciclo acelerado, não iniciar novo incremento enquanto houver PR crítico bloqueado sem decisão.
