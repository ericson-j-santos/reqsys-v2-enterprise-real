# Governance Policy — Operational Health Dashboard

## Política

Todo ciclo acelerado deve manter:

- fila curta de PRs;
- PRs obsoletos fechados;
- PRs críticos com score >= 85;
- CI verde antes de merge;
- separação clara entre governança, experimento e produção.

## Aplicação

Esta política é referência para decisões operacionais de PR.
