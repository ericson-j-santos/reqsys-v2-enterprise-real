# Rollout — Operational Health Dashboard Governance

## Fase 1 — Review-only

- Versionar documentos.
- Versionar workflow.
- Gerar artifact de evidência.
- Validar em PR.

## Fase 2 — Gerador automático

- Gerar dados reais de PRs.
- Gerar score real.
- Gerar dashboard HTML.
- Publicar artifact navegável.

## Fase 3 — Operação consolidada

- Usar dashboard como entrada de decisão operacional.
- Separar backlog experimental e backlog produtivo.
- Reduzir PRs obsoletos recorrentes.
