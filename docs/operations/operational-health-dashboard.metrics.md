# Metrics — Operational Health Dashboard Governance

## Métricas planejadas

| Métrica | Definição |
|---|---|
| `open_pr_count` | Total de PRs abertos |
| `draft_pr_count` | Total de PRs em draft |
| `blocked_pr_count` | PRs com conflito ou CI falha |
| `stale_pr_count` | PRs antigos ou defasados |
| `merge_readiness_score` | Score objetivo de prontidão para merge |
| `conflict_risk` | Risco estimado de conflito |
| `ci_health_trend` | Tendência de saúde dos checks |
