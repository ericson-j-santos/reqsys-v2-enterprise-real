FIGMA · Drill-down Flow

Analytics navegável · Padrão Ouro

Fluxo de drill-down — Dashboard ReqSys

1. Indicador

Card operacional com semáforo e valor consolidado no dashboard.

→

2. Clique

Usuário aciona card ou etapa do pipeline. Hint “Abrir analítico”.

→

3. Analítico filtrado

Rota SPA com query preservada: status, erro, estado crítico.

→

4. Ação operacional

Detalhe, exportação, correlation_id, reprocessamento ou issue.

Exemplos de rotas filtradas

/requisitos?status=em_analise

/painel-integracao?status=erro

/pipeline?status=error

/analytics → hub executivo com runtime cards

/monitoramento-operacional?estado=vermelho
