# Analytics Drill-down

Codigo: AI-GOV-FE-001

## Decisao

Sempre que fizer sentido, cards, graficos, indicadores e tabelas devem permitir abertura do analitico correspondente ja filtrado.

## Aplicacao no ReqSys

- Dashboards executivos.
- Paineis operacionais.
- Indicadores de backlog.
- Indicadores de qualidade.
- Monitoramento de integracoes.

## Regras

- Todo indicador deve ter regra de calculo clara.
- Todo drill-down deve preservar filtros aplicados.
- Toda tela analitica deve expor origem, data e criterio do dado.
- A interface deve ser responsiva.
