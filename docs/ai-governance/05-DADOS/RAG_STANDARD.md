# RAG Standard

Codigo: AI-GOV-DATA-001

## Objetivo

Padronizar arquitetura RAG corporativa.

## Regras

- Toda resposta deve possuir fonte rastreavel quando aplicavel.
- Dados devem possuir controle de acesso.
- Logs devem respeitar privacidade.
- Respostas sem base confiavel devem indicar baixa confianca.
- Ambientes devem ser segregados.

## Aplicacao

- ReqSys IA
- GovBI IA
- analytics conversacional
- documentacao viva
- busca semantica
