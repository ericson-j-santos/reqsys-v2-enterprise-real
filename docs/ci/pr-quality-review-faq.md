# FAQ — PR Quality Review

## O CodeRabbit foi removido?

O repositorio deixa de depender operacionalmente do CodeRabbit. A remocao do GitHub App, quando instalado, e feita nas configuracoes do GitHub.

## O novo workflow aprova PR sozinho?

Nao. Ele apenas gera evidencia e falha em risco critico objetivo.

## Warning bloqueia merge?

Nao por padrao. Warning fica visivel para decisao humana.
