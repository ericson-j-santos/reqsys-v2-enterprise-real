# Escopo de segurança — PR Quality Review

## Verificações objetivas

- nomes de arquivos potencialmente sensíveis;
- caminhos críticos;
- workflows alterados;
- ausência de testes quando há código;
- tamanho do PR.

## Limites

O workflow não substitui revisão humana de segurança, análise de secrets reais, SAST completo ou aprovação de produção.
