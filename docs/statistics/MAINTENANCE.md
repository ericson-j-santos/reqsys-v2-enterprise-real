# Manutenção — Aba Estatísticas

## Rotina recomendada

- Revisar mensalmente fórmulas e fontes.
- Validar indicadores sem evidência.
- Remover ou corrigir métricas sem uso decisório.
- Atualizar TTL de fontes externas.
- Revisar riscos e limitações antes de cada release.

## Evolução técnica

- Migrar fonte local para backend real.
- Adicionar histórico temporal.
- Adicionar exportação CSV/JSON.
- Adicionar drill-down com link para evidências.
- Adicionar integração com Power BI/GovBI IA quando houver base real.
