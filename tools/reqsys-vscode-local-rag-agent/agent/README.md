# ReqSys Agent

Agente local Python em modo somente leitura para indexação governada e consulta do workspace.

## Comandos

```bash
python -m reqsys_agent.cli health
python -m reqsys_agent.cli index --workspace <workspace>
python -m reqsys_agent.cli ask --workspace <workspace> --question "pergunta"
python -m reqsys_agent.cli governance --workspace <workspace>
```
